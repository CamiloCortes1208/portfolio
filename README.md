## Portafolio Profesional

Este es mi portafolio profesional desarrollado con Next.js y desplegado en Vercel. El proyecto presenta mis habilidades, experiencia y proyectos de forma moderna, responsiva y optimizada para rendimiento.

## Características

- Diseño responsive (adaptado a móvil y escritorio)
- Interfaz moderna con animaciones suaves en escritorio
- Navegación por secciones tipo SPA
- Componentización reutilizable en React
- Optimización de rendimiento con Next.js
- Despliegue continuo en Vercel

## Tecnologías utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel (deploy)
